#Plan B Landing page
==================


##Dependencies:
- grunt
- sass
- bower

##Grunt Tasks.


```
 grunt
```
Runs hosting


```
 grunt dev
```
build page for development

```
 grunt re-dev
```
remove previous build and re-build page for development

```
 grunt prod
```
Build pages, css and assets for production

```
 grunt re-prod
```
Remove previous build and re-build pages for production
